# Harsh Pandey — Portfolio 🌐💼

A **fast, modern, and fully responsive personal portfolio website** built with pure HTML, CSS, and JavaScript — no frameworks, no fluff. Showcasing projects, skills, certificates, and contact info with smooth animations and a clean UI.

## 🎯 Features

- ⚡ Blazing fast — zero frameworks, pure vanilla JS
- 📱 Fully responsive across all screen sizes
- 🎨 Smooth scroll animations via ScrollReveal
- 🗂️ Filterable certificates section by category
- 💼 Projects showcase with live links
- 🧠 Dynamic skills section loaded from `skills.json`
- 📬 Direct contact section with social links
- 🔗 Custom 404 error page

## 📁 Project Structure

```
portfolio/
├── index.html          # Main portfolio page
├── 404.html            # Custom 404 page
├── assets/
│   ├── css/
│   │   ├── style.css       # Core styles
│   │   ├── responsive.css  # Media queries
│   │   └── projects.css    # Projects section styles
│   ├── js/
│   │   ├── main.js         # Core interactivity & animations
│   │   └── projects.js     # Projects rendering logic
│   └── images/             # All images and icons
├── data/
│   ├── projects.js     # Projects data
│   └── skills.js       # Skills data
└── projects/           # Individual project pages
```

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/harshpandeyz/portfolio.git

# Navigate into the folder
cd portfolio

# Open in browser (no build step needed!)
open index.html
```

> Or just drag `index.html` into your browser — it works instantly.
